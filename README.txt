A simple CLI script to open a video in Youtube.
Uses core python libraries.
Allowed inputs : 
1. "python3 main.py <searchterm>"
2. "python3 main.py -n <number> <searchterm>"
3. "./main.py <searchterm>"
4. "./main.py -n <number> <searchterm>"
For (3) and (4) you need to use the "chmod +x <filename>.py" command to make it executable.
